Z Decimal for Java

Java utility classes to process IBM System Z packed and zone decimal data.

Download

Documentation in the Javadoc

The demo program screenshot

Installation

  • Place the zdecimal jar file in your CLASSPATH
  • Extract the files using the jar xf command
  • The doc directory contains the javadoc files.

An example program of demo.java is included. The source code is shipped in the jar file.
